*🐦🐦 Flappy Bird (My First Python Project)🐦🐦*

This is my very first programming project It is famous game name flappy bird i have created a clone or copy of it
I made this project to learn how to make games how to make things move with gravity and how to check when things crash into each other

---


**🚀 Features**
- *Gravity: The bird falls down automatically and jumps up when you press a key
- *Moving Pipes: Green pipes scroll across the screen 
- *Score Tracking : tracks your current score
- *High Score System : it also saves your highscore for your current game
- *Developer Name: My name is cleanly drawn in white text inside the bottom blue bar

---

**🛠️ Tech Stack**
- Python
- Pygame

**📦 Installation**

- Download the code
- open terminal and write following commands
- pip install pygame
- python "main.py" (to run the game)

---

**🎯 How to Play**
- Press SPACE to jump
- Avoid hitting pipes
- Try to beat your high score
